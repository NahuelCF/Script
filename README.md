# Script

### Uso

* Lo primero a hacer es clonar el repositorio en su maquina utilizando Git Bash o desde la terminal de VSCODE.

* Una vez clonado nos dirijimos a la carpeta con el comando "cd ./script" y ejecutamos "npm install" para instalar las dependencias.

* Hecho esto, solo nos queda ejecutar el comando "node .\src\controller\script.js > script.log 2>&1" e introducir el seller_id, o si se quiere ingresar mas de un seller_id se tienen que ingresar separados por coma (179571326,179572026), y pulsar intro.

* Finalmente se nos crea el archivo script.log con los items del seller_id ingresado.
